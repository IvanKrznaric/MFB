# 5. Domains and limits of functions

## 5.1 Domain of a function

:::{tip} Definition
:icon:false
**Domain** of the function $f$ is the set of all real numbers $x$ for which that function is defined. Such as set is denoted by $\mathcal{D}_f.$
:::

\end{itemize}

:::

2.1 Introduction¶

In the previous chapter we learned that matrices are very similar to numbers - you know how to add two matrices, how to subtract them and you know how to multiply two matrices. The only operation we haven’t mentioned so far is divison - if you have two matrices AAA and BBB, what would it mean to divide matrix AAA by the matrix BBB?

In order to answer that quesion, let’s see what does it mean to divide two numbers: for example, if we are given numbers 5 and 7, then to divide number 5 by 7 we have to multiply 5 by 17,\frac{1}{7},71​, i.e.

5÷7=5⋅17.5 \div 7 = 5 \cdot \frac{1}{7}.5÷7=5⋅71​.

What is the connection between numbers 7 and 17?\frac{1}{7}?71​? Well, if we multiply them in any order, we get number 1 as a result:

7⋅17=17⋅7=17 \cdot \frac{1}{7} = \frac{1}{7} \cdot 7 = 17⋅71​=71​⋅7=1

This also means that the reciprocal value or the inverse of the number 7 is 17.\frac{1}{7}.71​.

So, we can see that there are two parts that are crucial when dividing number aaa by the number bbb:

  1. Find the inverse 1b\frac{1}{b}b1​ of the number bbb

  2. Multiply aaa and 1b\frac{1}{b}b1​

We can use this approach to define “divison” of matrices. If we would like to divide matrix AAA by the matrix BBB, firstly we need to define what would be the inverse of the matrix BBB and then we would simply need to multiply AAA by that inverse.

Definition

Let AAA be a n×nn \times nn×n matrix. We say that a n×nn \times nn×n matrix XXX is the inverse of the matrix AAA if

A⋅X=X⋅A=I.A \cdot X = X \cdot A = I .A⋅X=X⋅A=I.

In that case, we say that the matrix AAA is invertible or regular. The inverse XXX of the matrix AAA is denoted by A−1.A^{-1}.A−1.

Remark

Of course, the inverse of the matrix AAA is denoted by A−1A^{-1}A−1 for the same reason why the reciprocal value of the number 7 is denoted by 7−1.7^{-1}.7−1.

So, if we would like to divide the matrix AAA by the matrix BBB, we would need to find the inverse B−1B^{-1}B−1 of the matrix BBB and then compute A⋅B−1.A \cdot B^{-1}.A⋅B−1. However, here we stumble upon a problem - not all matrices are invertible.

2.2 Determinant of a matrix¶

Because not every matrix is invertible, we need a criterion that will allow us to check whether or not a given matrix has an inverse. For that, we will be using the determinant. The determinant is a number that is associated to every square matrix. The determinant of a matrix AAA is denoted by ∣A∣\lvert A \rvert∣A∣ or by det(A).\text{det}(A).det(A). The determinant of a 2×22 \times 22×2 matrix is given by a simple formula

∣abcd∣=ad−bc.\begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc.∣∣​ac​bd​∣∣​=ad−bc.
Exercise 2.2

In order to find determinant of matrices of higher order, we will use two methods:

  • Laplace expansion

  • Elementary transformations

Note

Although you can use each method separately, in most of the problems it will be best if you use both of them together to find the determinant.

Laplace expansion¶

Algorithm (Laplace expansion)

  1. Choose a row or a column (preferably one with the most zeros)

  2. Determine the signs of its elements

  3. Run through the chosen row/column and expand

Main idea of the Laplace expansion

Notice the main point about Laplace expansion: with each application of a Laplace expansion, we are reducing the order of the matrix whose determinant we need to calculate and hence we are reducing the complexity of our calculations.

Elementary transformations¶

As we have seen, Laplace expansion is a very useful method of finding the determinant of a matrix that has a lot of zeros along some row or a column, however what if our has matrix doesn’t have any zeros? In that case, we will use elementary transformations to get the zeros in our matrix so that we can use the Laplace expansion to find the determinant. To use an elementary transformation simply means to multiply a row of a matrix by some number and adding it to some other row. Obviously, the end-goal of performing elementary transformations is to get zeros along some column of the given matrix so that we can use Laplace expansion to calculate the determinant.

Main idea of elementary transformations

Notice the main point of elementary transformations: when a matrix has a lot of zeros, finding the determinant is easy because we can use the Laplace expansion. If the matrix doesn’t have a lot of zeros, we are using elementary transformations to reduce the complexity of the matrix and to obtain something very simple that we know how to work with.

The most important theorem when talking about the determinant is the Binet-Cauchy theorem.

Theorem 1 (Binet-Cauchy)

Let A,BA,BA,B be two n×nn \times nn×n matrices. Then det(A⋅B)=det(A)⋅det(B).\text{det}(A \cdot B) = \text{det}(A) \cdot \text{det}(B).det(A⋅B)=det(A)⋅det(B).

2.3. Matrix inversion¶

As we have mentioned in the introduction, the main point of the determinant is to give us a criterion for checking whether or not a matrix has an inverse or not. That criterion is the following:

Criterion
det(A)≠0  ⟹  A has an inversedet(A)=0  ⟹  A doesn’t have an inverse\begin{split} \text{det}(A) \neq 0 &\implies A \text{ has an inverse} \\ \text{det}(A) = 0 &\implies A \text{ doesn't have an inverse} \end{split}det(A)=0det(A)=0​⟹A has an inverse⟹A doesn’t have an inverse​

Next thing we would like to know is how to explicitly compute the inverse of a matrix.

Definition

Let A=[aij]∈MnA = [a_{ij}] \in M_nA=[aij​]∈Mn​ be a square matrix.

  • The minor of an element aija_{ij}aij​ is the determinant of a matrix obtained by deleting the iii-th row and the jjj-th column of the matrix AAA. That number is denoted by MijM_{ij}Mij​.

  • The cofactor matrix is a n×nn \times nn×n matrix C=[cij],C = [c_{ij}],C=[cij​], where

cij=(−1)i+jMijc_{ij} = (-1)^{i+j}M_{ij}cij​=(−1)i+jMij​

  • The adjoint of the matrix AAA is the matrix A∗=CT.A^{\ast} = C^T.A∗=CT.

Finally, the inverse of the matrix AAA is given by the formula

A−1=1det(A)A∗.A^{-1} = \frac{1}{\text{det}(A)}A^\ast.A−1=det(A)1​A∗.

2.4 Cramer’s systems¶

One of the main application of matrices we are going to see is revolving around solving system’s of linear equations.

Example

The system

{2x−5y=153x+y=31.\begin{cases} 2x-5y = 15 \\ 3x+y = 31. \end{cases}{2x−5y=153x+y=31.​

can be written in the matrix form as

[2−531]⋅[xy]=[1531]\begin{bmatrix} 2 & -5 \\ 3 & 1 \end{bmatrix} \cdot \begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 15 \\ 31 \end{bmatrix}[23​−51​]⋅[xy​]=[1531​]

In general, every system can be written in matrix form as

AX=B,AX = B,AX=B,

where AAA is the matrix of coefficients, XXX is the column-matrix of unknowns and BBB is the column-matrix of the right-hand side of the system.

Definition

We say that a system AX=BAX = BAX=B of nnn equations with nnn unknows is Cramer’s system if det(A)≠0.\text{det}(A) \neq 0.det(A)=0.

Cramer’s system are particularly easy to solve because their solutions are given by the formula

xi=DiD,i=1,…,nx_i = \frac{D_i}{D}, \quad i = 1, \dots, nxi​=DDi​​,i=1,…,n

where D=det(A)D = \text{det}(A)D=det(A) and DiD_iDi​ is the determinant of the matrix obtained by replacing the iii-th column of the matrix AAA by the column-matrix BBB.
